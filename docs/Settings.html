Class: Settings

Settings(optionsopt)

Manage settings for how martian performs API requests.

Constructor

new Settings(optionsopt)

Create a new martian Settings object.
Parameters:
Name Type Attributes Description
options Object <optional>
 The options for the martian settings object.
Properties
Name Type Attributes Description
host String <optional>
 The URI of the mindtouch site that is hosting the API to access.
queryParams Object <optional>
 An object mapping query parameter keys with values. Keys and values will both be converted to strings.
headers Object <optional>
 An object mapping HTTP header keys with values. Keys must be strings, and values will be converted to strings.
token String | function <optional>
 A token to allow API access. This will populate the "X-Deki-Token" header. If a function is supplied, it should return the desired token as a string.
origin String <optional>
 The origin of the API calls.
